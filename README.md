# 🔍 Job Web Hunt — Advanced Career Connectivity Platform

> A comprehensive job board application that bridges the gap between talent and opportunity with smart filtering and seamless application tracking.

---

## 📌 Problem Statement
Job seekers often struggle with cluttered interfaces and irrelevant job postings, while employers find it difficult to manage high volumes of applications. There is a need for a streamlined, organized, and responsive platform that simplifies the recruitment process.

## 💡 Solution Overview
**Job Web Hunt** provides a clean, user-centric interface for searching and posting jobs. Built with a focus on performance and usability, it features real-time search, category-based filtering, and a dedicated dashboard for users to monitor their application journey.

## 🛠️ Tech Stack

| Category | Tools |
| :--- | :--- |
| **Frontend** | `React` • `Tailwind CSS` • `Lucide React` |
| **Backend** | `Node.js` • `Express` • `MongoDB` |
| **Deployment** | `Vercel` (Frontend) |
| **State Management** | `React Hooks` / `Context API` |

## ✨ Key Features
- **🎯 Smart Search:** Instant job searching by titles, companies, or keywords.
- **📑 Dynamic Filtering:** Filter jobs by category, location, and employment type.
- **📈 Application Tracking:** A dedicated space for users to see their applied jobs and status.
- **💼 Employer Dashboard:** Easy tools for posting jobs and managing incoming candidates.
- **📱 Responsive UI:** A pixel-perfect experience across all mobile and desktop devices.
